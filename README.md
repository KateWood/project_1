# Tic-Tac-Toe

This is a fully functional Tic-Tac-Toe game which can be played on the same computer or on two different devices.

#### Technologies Used

* HTML

* CSS

* JavaScript

* Angular.js

* Firebase

#### See My Tic-Tac-Toe in Action

[Tic-Tac-Toe on Firebase](https://tictactoekate.firebaseapp.com/)
